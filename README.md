## How to use:
- Compatible with:
	- Linux(Desktop)
	- Termux(Android)
- Requirements:
	- Python3
	- pip3
	- Git
- pip packages:
	- termcolor
	- colorama
	- requests
	- bs4
	- html5lib
	- phonenumbers
	- argparse
	- urllib3
- Commands:
```
git clone https://github.com/AbirHasan2005/OPRecon
cd OPRecon
python3 -m pip install -r requirements.txt
bash run.sh
```
